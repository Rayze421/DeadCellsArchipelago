This is an archipelago mod for Dead Cells, using Core Modding (https://github.com/dead-cells-core-modding/core)

To set up this mod you'll need first to follow Core Modding installation guide.

Then in the coremod directory you'll need to put the mods directory in the .zip from the latest release of DeadCellsArchipelago.
In the end, the path should look like 'Dead Cells\coremod\mods\DeadCellsArchipelago'

For now, if you want to change the connection data, it's in 'Dead Cells\coremod\mods\DeadCellsArchipelago\conf.json'

If you want to launch the modded game, you'll find it at 'Dead Cells\coremod\core\host\startup\DeadCellsModding.exe'

You should start your game from a new save, else it might corrupt what you've done.

The DLCs aren't mandatory, but I advise you to have Return To Castlevania, else you might be softlock until you find the vine rune.
